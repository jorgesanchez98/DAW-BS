Mi primer año en el Tec

Después de dos largos semestres y un verano más corto de lo que solían ser, finalmente puedo decir que ya sobreviví el inicio de la universidad, ahora solo queda terminarla. El primer año fue duro, por primera vez viví solo, lo que significaba tomar la responsabilidad sobre mi mismo, pero también era tener una libertad mayor, sin embargo a diferencia de lo que se podría pensar que eso significaría, en mi caso fue mayormente el desvelarme estupidamente terminando proyectos, llegar tarde a casa después de terminar de trabajar en equipo en el Tec o pasar días enteros haciendo tareas que debiste haber hecho hace semanas. La universidad no fue muy bonita el primer año, pero principalmente porque yo la hice así, muy posiblemente por la incertidumbre que sentía por ser el inicio de mi carrera y no estar completamente seguro si tome la decisión correcta. Ahora en mi segundo año, veo las cosas con diferente perspectiva, ahora realmente puedo decir que soy un adulto que toma las riendas de su vida y toma acciones para mejorar su futuro, gracias a la experiencia de ese primer año

Definiciones

Término Definición
URL Es la dirección específica de un cierto recurso dentro del lugar donde este almacenado
MIME Es un protocolo de la capa aplicación que permite transmitir datos multimedia y ayuda al cliente a saber que tipo de reproducto usar
Métodos HTTP Get-Recupera datos
HEAD-igual que GET pero sin el "body"
POST-Envía un recurso específico
PUT-Hacer cambios en el recurso
DELETE-Borrar un recurso
PATCH-Modificaciones parciales a un recurso[1]
XML Lenguaje de marcado diseñado para ser legible ambas por máquina y humano
XHTML Lenguaje de marcado extensión de HTML pero parte de la familia de XML
HTML5 La versión mas actualizada de HTML y sus estándares
JavaScript Lenguaje de programación usado para añadir dinamicidad a una página web
PHP Lenguaje de programación popular para el desarrollo web que funciona principalmente en el servidor[2]
JSP Es un asistente de servidor que cumple el rol de interfaz de usuario para una aplicación web Java[3]
Ajax Son técnicas de desarrollo web que usa multiples tecnologías para crear aplicaciones web asíncronas[4]

Preguntas a responder:

  • ¿Cuál es la diferencia entre Internet y la World Wide Web?
    Internet es el conjunto de computadoras y servidores conectados al rededor del mundo y World Wide Web es el espacio donde recursos web son accesador por medio de la Internet
  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
    Recibir y mandar información con el protocolo HTTP
  • ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?
    POST, para que esten escondidos los datos
  • ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?
    GET
  • Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?
    Que el proceso fue exitoso
  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?
    No, porque el error fue causado al cliente al intentar ingresar un link erroneo o porque así fue diseñado por el desarrollador
  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?
    Si, porque el error esta en el servidor
  • ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.
    Significa que funciona pero su uso no es recomendado, font y strike son dos atributos depreciados
  • ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?
    HTML5 es la versión actualizada de HTML4, tiene más restricciones en cuanto a sintaxis y uso de tags, además agrega nuevas funciones como geolocalización o video directamente en el código
  • ¿Qué componentes de estructura y estilo tiene una tabla?
    Separa el texto en header y cuerpo, columndas y filas, además los headers estan en negritas
  • ¿Cuáles son los principales controles de una forma HTML5?
    Control de fecha y hora, control de color, barra de busqueda, lista
  • ¿Qué tanto soporte HTML5 tiene el navegador que utilizas?
    Tiene 515 puntos de 555
  • ¿Cuál es el ciclo de vida de los sistemas de información?
    Planificación, análisis, diseño, planificación, implementación, pruebas, instalación, uso y mantenimiento
  • Comunicación, requisitos del sistema, estudio de factibilidad, análisis del sistema, diseño de software, códigos, pruebas, integración, implementación, operaciones, mantenimiento y disposición[5]

Preguntas Lab 3:

  • Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?
    No abusar de el pues puede causar que tu código sea dificil de mantener y actualizar
  • Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?
    Para evitar que el contenido se disuelva en el fondo y deje de ser legible
  • Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?
    Tener en cuenta la disponibilidad que tendra tu aplicación web en diferentes dispositivos, más si quieres que este optimizado para celulares y tabletas
  • ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?
    Porque reduce la cantidad de bits que tienen que llegar al usuario
