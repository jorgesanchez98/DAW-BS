¿Que es una base de datos?
Es una colección de de datos interrelacionados optimizados para evitar la menor cantidad de repeticiones de datos
¿En que casos es conveniente usar bases de datos?
Cuando la cantidad de información almacenada es muy grande y la cantidad de gente que desea accesar a ella tambíen lo es
¿Qué es un sistema de gestión de base de datos - DBMS?
Es un sistema diseñado para el almacenamiento y extracción de datos en una base de datos, garantizando que pueda ser utilizado por todos los interesados de manera segura
